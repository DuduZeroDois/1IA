# 🚀 1IA

Uma inteligência artificial desenvolvida para explorar automação, processamento inteligente e interação com usuários de forma simples e eficiente.

---

## 📌 Sobre o projeto

O **1IA** é um projeto focado em desenvolvimento de soluções utilizando conceitos de Inteligência Artificial.  
O objetivo é criar uma base prática e escalável para experimentos, automações e aplicações inteligentes.

---

## ✨ Funcionalidades

- ⚡ Processamento automatizado
- 🤖 Interação inteligente
- 📊 Estrutura organizada para expansão
- 🔧 Fácil configuração e execução
- 💡 Integração com IA Gemini

---

## 🛠️ Tecnologias utilizadas

- Machine Learning

# 🚀 Como acessar e executar o projeto

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/DuduZeroDois/1IA.git
```

---

## 2️⃣ Entre na pasta do projeto

```bash
cd 1IA
```

---

## 3️⃣ Instale as dependências

```bash
pip install -r requirements.txt
```

---

# 🔑 Como conseguir a API Key do Gemini

## 1️⃣ Acesse o Google AI Studio

👉 https://aistudio.google.com/

---

## 2️⃣ Faça login com sua conta Google

Use qualquer conta Google válida.

---

## 3️⃣ Crie uma API Key

- Clique em **"Get API Key"**
- Depois em **"Create API Key"**
- Copie a chave gerada

---

## 4️⃣ Configure a chave no projeto

Crie um arquivo chamado `.env` na raiz do projeto.

Exemplo:

```env
GEMINI_API_KEY=sua_chave_aqui
```

---

# ▶️ Como executar

Depois de configurar tudo:

```bash
python main.py
```
